# Entity Relationship Diagram (ERD)

Folder ini memuat ERD final dari sistem basis data
Fotocopy & Print “Focus”.

ERD menggambarkan:
- Entitas: Pelanggan, Kasir, Barang/Layanan, dan Transaksi
- Atribut tiap entitas
- Hubungan antar entitas beserta kardinalitasnya

ERD berfungsi sebagai blueprint sistem basis data.
Dari diagram ini, struktur tabel, primary key, dan foreign key
diturunkan lalu diimplementasikan ke MySQL menggunakan MySQL Workbench.

Dengan adanya ERD, rancangan basis data menjadi:
- Lebih terstruktur
- Mudah dipahami
- Minim kesalahan saat implementasi

# Poster Proyek

Folder ini berisi poster proyek UAS Mata Kuliah **Pemrograman Basis Data** dengan tema  
**“Perancangan dan Implementasi Basis Data Sistem Transaksi Fotocopy & Print ‘Focus’”**.

Poster ini dirancang sebagai media visual ringkas yang merangkum seluruh proses
perancangan basis data, mulai dari permasalahan awal hingga implementasi ke dalam DBMS.

## Isi Poster

Poster menampilkan beberapa komponen utama:

- **Ringkasan Masalah**  
  Menjelaskan kondisi awal pengelolaan data transaksi yang masih sederhana,
  tidak terstruktur, dan rawan pengulangan data.

- **Perancangan Basis Data**  
  - Hubungan antar entitas  
  - Derajat kardinalitas:
    - Pelanggan – Transaksi : 1 : N  
    - Kasir – Transaksi : 1 : N  
    - Barang/Layanan – Transaksi : N : N  
  - Visualisasi ERD sebagai blueprint sistem

- **Implementasi Database**
  - DDL: pembuatan database dan tabel  
  - DML: pengisian data  
  - TCL: pengelolaan transaksi  

- **Query Wajib**
  - JOIN  
  - GROUP BY & Agregasi  
  - HAVING  

- **Tujuan Proyek**
  - Merancang basis data relasional yang terstruktur  
  - Melakukan normalisasi hingga 3NF  
  - Membuat ERD dan relasi tabel  
  - Mengimplementasikan database menggunakan MySQL  

Poster ini berfungsi sebagai representasi visual dari laporan lengkap dan repository,
sehingga audiens dapat memahami gambaran sistem hanya dalam satu tampilan.

Poster digunakan sebagai media presentasi dan dokumentasi ringkas proyek.

# Laporan Proyek

Folder ini berisi laporan resmi Proyek UAS Mata Kuliah
Pemrograman Basis Data.

Laporan mencakup:
- Latar belakang dan tujuan proyek
- Analisis data mentah nota fotocopy
- Proses normalisasi (1NF, 2NF, 3NF)
- Perancangan relasi dan ERD
- Implementasi basis data di MySQL
- Penggunaan DDL, DML, dan TCL
- Pengujian query (JOIN, GROUP BY, HAVING, Subquery)
- Kesimpulan dan saran pengembangan

Dokumen ini menjadi bukti akademis proses perancangan sistem basis data
yang dilakukan secara bertahap dan terstruktur.

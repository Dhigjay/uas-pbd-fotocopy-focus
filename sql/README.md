# Script SQL

Folder ini berisi file SQL yang digunakan untuk membangun sistem basis data
Fotocopy & Print “Focus”.

Script mencakup:
- Pembuatan database
- Pembuatan tabel (DDL)
- Penentuan Primary Key dan Foreign Key
- Pengisian data awal (DML)
- Mengontrol Transaksi pada Database seperti `START TRANSACTION` `COMMIT` `ROLLBACK` `SAVEPOINT` `ROLLBACK TO SAVEPOINT` (TCL)
- Beberapa Query (JOIN, GROUP BY & AGREGASI, HAVING, SUBQUERY)

File ini merupakan implementasi nyata dari ERD dan relasi antar tabel.
Dengan menjalankan script ini pada MySQL, seluruh struktur basis data
dapat langsung terbentuk sesuai rancangan pada laporan.

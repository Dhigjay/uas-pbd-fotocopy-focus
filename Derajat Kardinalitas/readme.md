# Derajat Kardinalitas

Folder ini berisi visualisasi derajat kardinalitas pada sistem basis data
Fotocopy & Print “Focus”.

Derajat kardinalitas digunakan untuk menjelaskan banyaknya hubungan
antar entitas, seperti:
- 1 : 1 (one ke one)
- 1 : N (oneu ke many)
- N : N (many ke many)

Dalam proyek ini, kardinalitas yang diterapkan antara lain:
1. Pelanggan → Transaksi (1 : N)  
   Satu pelanggan dapat melakukan banyak transaksi.
2. Kasir → Transaksi (1 : N)  
   Satu kasir dapat melayani banyak transaksi.
3. Barang/Layanan → Transaksi (N : N melalui tabel transaksi)

Gambar pada folder ini menjadi dasar dalam menentukan struktur relasi
antar tabel agar sesuai dengan kebutuhan sistem dan menjaga konsistensi data.
